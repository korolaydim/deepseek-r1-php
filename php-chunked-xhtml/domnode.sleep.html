DOMNode::__sleep

(PHP 8 >= 8.1.25)

DOMNode::__sleep — Forbids serialization unless serialization methods are implemented in a subclass

Description

public DOMNode::__sleep(): array

Forbids serialization unless serialization methods are implemented in a subclass.

Parameters

This function has no parameters.

Return Values

The method always throws an exception.

Errors/Exceptions

Throws an Error exception when called.
